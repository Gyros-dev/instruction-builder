# KASSIR.RU — конструктор инструкций

Статический проект без сервера и базы данных. Подходит для размещения на GitHub Pages.

## Возможности

- создание пошаговых инструкций;
- форматирование текста каждого шага;
- добавление скриншотов, подписей и подсказок;
- изменение порядка шагов;
- блоки «Перед началом», «Важно», «Результат» и «Если не получилось»;
- живой предпросмотр;
- автоматическое сохранение в браузере;
- сохранение и загрузка черновика JSON;
- экспорт автономного HTML;
- печать и сохранение в PDF.

## Запуск локально

Откройте `index.html` в браузере. Для корректной загрузки логотипа лучше запустить локальный сервер:

```bash
python3 -m http.server 8000
```

Затем откройте `http://localhost:8000`.

## GitHub Pages

1. Создайте репозиторий.
2. Загрузите все файлы проекта в ветку `main`.
3. Откройте **Settings → Pages**.
4. Выберите **Deploy from a branch**, ветку `main`, папку `/ (root)`.
